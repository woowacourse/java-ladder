## 입력
- [x] 참여자 이름 입력
  - [x] 최대 5글자까지 가능
  - [x] 공백인 경우 예외처리
  - [x] 중복인 경우 예외처리
- [x] 최대 사다리 높이 입력
  - [x] 숫자가 아닌 경우 예외처리 
  - [x] 0이하인 경우 예외처리

## 출력
- [x] 사다리 결과 출력
  - 사람 이름을 5자 기준으로 출력

  - 다리가 없는 경우 |     |
