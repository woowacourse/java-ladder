# java-ladder
사다리타기 미션 저장소
## 기능과 예외상황
##### [기능]
  1. 사람 이름을 입력 받는다.
    
     ##### [예외]
     1. 5글자 초과
     
  2. 입력 받은 이름을 쉼표(,) 기준으로 구분한다.
  ```
  "denis,whale,pobi" -> "[denis, whale, pobi]"
  ```
  
  3. 경품을 입력 받는다.
    
     ##### [예외]
     1. 5글자 초과
  
  4. 최대 사다리 높이를 입력받는다.
  
     ##### [예외]
     1. 0 이하의 값
  
  5. 사다리를 만든다.
  
  6. 사다리를 출력한다.
  
  7. 결과를 원하는 사람의 이름을 입력 받는다.
  
     #### [예외]
     1. 게임에 참가한 사람의 이름이 아닌 경우
  
  8. 결과를 출력한다.
    


# 문자열 계산기
## 기능과 예외상황
##### [기능]
  1. 커스텀 구분자 지정
  
     ```
     "//;\n" -> ";" 
     ``` 
     
     ##### [예외]
     1. 커스텀 지정자 하나 초과
     2. 커스텀 지정자에 숫자가 들어가는 경우

  2. 구분자를 기준으로 Split
    
     ```
     "1,2,3" -> "[1,2,3]"
     "1:2:3" -> "[1,2,3]"
     "1,2:3" -> "[1,2,3]"
     "1;2;3" -> "[1,2,3]"
     ``` 

     ##### [예외]
     1. 구분자가 연속되는 경우
     2. 숫자가 연속되는 경우
     3. 구분자 리스트외에 값이 들어온 경우
     
  3. 숫자의 합을 반환
     
     ```
     "[1,2]" -> "3"
     "[1,2,3,4]" -> "10"
     ```
    
     ##### [예외]
     1. 숫자 이외의 값 또는 음수
