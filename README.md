# java-ladder

사다리 타기 미션 저장소

## 기능 목록

### 입력

- [X] 이름들
    - [X] 예외
        - [X] 이름이 1-5자 범위를 벗어난 경우
        - [X] 이름이 중복된 경우
- [X] 높이
    - [X] 예외
        - [X] 높이가 자연수가 아닌 경우

### 로직

- [X] 사다리 생성
    - [X] 가로줄은 무작위로 생성
- [X] 예외가 발생할 경우 다시 입력

### 출력

- [X] 이름들
- [X] 사다리

## 예시

```
참여할 사람 이름을 입력하세요. (이름은 쉼표(,)로 구분하세요)
pobi,honux,crong,jk

최대 사다리 높이는 몇 개인가요?
5

실행결과

pobi  honux crong   jk
    |-----|     |-----|
    |     |-----|     |
    |-----|     |     |
    |     |-----|     |
    |-----|     |-----|

```

각 칸은 세로줄을 의미하고, 1과 -1의 쌍은 가로줄을 의미합니다.  

즉 1은 오른쪽 방향을, -1은 왼쪽 방향을, 0은 방향이 없음을 의미합니다.
